Autonomous Navigation System

This project is an embedded C application that transforms a Roomba into a Mars Rover, navigating its environment while avoiding obstacles in real-time. Using a combination of IR sensors, sonar (Ping) sensors, and bumper sensors, the robot triggers pre-programmed functions that analyze the sensor data and adjusts the robot's path. This enables it to navigate through an obstical-ridden path without human intervention.

Additionally, I modified a Python GUI that takes the robot's sensor data and uses it to map the terrain surrounding the robot. This allows the robot to be observed without physically looking at it, similar to the mars rover.

Technical Highlights:

Languages: Embedded C, Python

Embedded Systems: Implemented data collection from IR, sonar, and bumper sensors to detect and avoid obstacles.

GUI & Visualization: Designed a Python-based interface for live terrain mapping and navigation feedback.
